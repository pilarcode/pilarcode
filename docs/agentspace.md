### ◻️ Agentspace

Agent Space, es una plataforma donde puedes interactuar con todos tus datos y agentes para acabar con los silos de información mediante las mejores técnicas de búsqueda, genAI y Agentes. Muchas empresas actualmente usan muchas herramientas basadas en AI pero están fragmentadas pero la cosa se complica cuando quieres combinar muchas fuentes de datos y distintas herramientas de AI.

  <img width="745" height="420" alt="image" src="https://github.com/user-attachments/assets/841239b4-90f0-4794-ae16-e97cbe32d74f" />

Agentspace, es un producto de software como servicio totalmente gestionado, que actua como un centro de información centralizado para una organización.  Es como si fuera una búsqueda en Google pero con todo lo que ocurre de una  empresa. 

 <img width="1701" height="916" alt="image" src="https://github.com/user-attachments/assets/02e35b26-af64-4122-8939-4a3f28d6d98f" />
 
Se pueden crear motores de búsqueda internos empresariales y agentes de búsqueda que trabajen con tus datos. Puedes acceder a distintas fuentes de datos mediante los conectores de agent space.

### ◻️ NotebookLM

  
  NotebookLM is a notebook where you can have conversations about your notes. NotebookLM is built with generative AI that helps you understand your notes and data sources, generate summaries, and answer specific questions about them. You can also create a podcast about
  your notes. If you need to work with more data , teams or customize the features you can work with NotebookLM Plus.

  However, if you work in a company and you need to access to jira or other data sources you can use NotebookLM Enterprise. You can import files from Microsoft Office. Plus, you can import data  from any source in agentspace search result. This includes fist party    sources like (gmail, cloud storage) and thir party sources like (atlassian, github,notion, salesforce and a bunch of others)

  <img width="1910" height="887" alt="image" src="https://github.com/user-attachments/assets/e3aea4ba-d152-495e-8cb6-a61bab0dea33" />


  You can enable NotebookLM Enterprise as a data source in Google Agentspace. This lets your users search the content of their notebooks using Agentspace apps.
